
# Smart Healthcare Management System

A Smart Healthcare Management System using Logistic Regression for predictive healthcare analytics. This project aims to digitalize patient data management and disease prediction for efficient healthcare services.

## Features
- User Registration & Login System
- Patient Information Management
- Disease Prediction using Logistic Regression (Python)
- Simple & Responsive Frontend using HTML
- Healthcare Data Handling

## Tech Stack
- Frontend: HTML, CSS
- Backend: Python (Flask)
- Machine Learning: Logistic Regression
- Database: File-based storage (can be upgraded)

## Installation

1. Clone the Repository
```bash
git clone https://github.com/aditya038/Smart-Healthcare-management-system.git
cd Smart-Healthcare-management-system
```

2. Install Dependencies
```bash
pip install -r requirements.txt
```

3. Run the Project
```bash
python app.py
```

4. Visit in Browser
```
http://localhost:5000
```

## Project Structure
```
├── templates/        → HTML Templates
├── static/           → CSS/JS Files
├── app.py            → Flask Application
├── model.pkl         → Trained ML Model
├── requirements.txt  → Python Dependencies
```

## How it Works
1. User Registers/Login
2. Fill Patient Details
3. Disease Prediction using Logistic Regression Model
4. Output Recommendation based on Prediction
